# Assets Directory

This directory contains static assets for the documentation:

- Images (screenshots, diagrams, logos)
- Icons and other graphics

## Usage

Reference assets in markdown with relative paths:

```markdown
![Screenshot](assets/screenshot.png)
```

## File Organization

```
assets/
├── images/           # Screenshots and photos
├── diagrams/         # Technical diagrams
└── icons/           # Icons and logos
```
